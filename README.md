## About the admin hub

The admin hub is an open source app that allows you to manage all aspects of your GetCandy store.

---

## Requirements
- PHP ^8.0
- Laravel 8+
- MySQL 5.7+ / PostgreSQL 9.2+
- GetCandy Core
